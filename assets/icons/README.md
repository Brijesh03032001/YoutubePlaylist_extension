# Icon Placeholder Files

This directory should contain the extension icons in the following sizes:

- **icon16.png** - 16x16 pixels (toolbar icon)
- **icon32.png** - 32x32 pixels (toolbar icon @2x)
- **icon48.png** - 48x48 pixels (extension management page)
- **icon128.png** - 128x128 pixels (Chrome Web Store)

## Creating Icons

### Option 1: Use an Icon Generator

Visit: https://favicon.io/ or https://realfavicongenerator.net/

1. Upload a base icon (512x512 recommended)
2. Generate all sizes
3. Download and place in this directory

### Option 2: Use Design Tool

Use Figma, Canva, or Photoshop:

1. Create a 512x512 canvas
2. Design your icon (use the ðŸ“š emoji or custom design)
3. Export as PNG in all required sizes:
   - 16x16
   - 32x32
   - 48x48
   - 128x128

### Icon Design Guidelines

**Recommended Design:**
- Use a book/playlist related icon
- Simple, recognizable design
- Works well at small sizes
- Matches the blue theme (#5B7FFF)
- Clear on both light and dark backgrounds

### Temporary Solution

Until you create proper icons, you can temporarily:
1. Take a screenshot of a blue square (any image editor)
2. Resize to 128x128, 48x48, 32x32, 16x16
3. Save as icon{size}.png in this directory

**Note:** The extension will still work without icons, but they're required for Chrome Web Store publication.
